# Introduction

Imagine you are asked to build:

- An online banking system
- A food delivery app
- A hospital management system
- An AI chatbot

Would you immediately start writing code?

Professional software teams don't.

They first understand **what to build, why to build it, how to build it, who will build it, how it will be tested, and how it will be maintained.**

That complete journey is called the **Software Development Life Cycle (SDLC).**

---

# What is SDLC?

**Software Development Life Cycle (SDLC)** is a structured framework that defines the stages involved in developing, delivering, operating, and maintaining software.

It transforms an idea into a reliable software product through a sequence of well-defined activities.

```text
Idea
  |
Requirements
  |
Design
  |
Development
  |
Testing
  |
Deployment
  |
Maintenance
```

- Improve planning

- Improve communication
- Simplify maintenance

# Benefits of SDLC

- Predictable development
- Better documentation
- Reduced defects
- Easier project tracking
- Higher software quality
- Easier collaboration
- Lower long-term cost

---

# Challenges

- Initial planning takes time
- Documentation requires effort
- Changes must be managed carefully
- Different projects need different SDLC models

# SDLC Models

The phases remain similar, but the execution differs.

Popular SDLC models include:

- Waterfall
- V-Model
- Incremental
- Iterative
- Spiral
- RAD
- Prototype
- Agile

These models will be covered in later lessons.

---

# Common Misconceptions

❌ SDLC means writing code.

✔ Coding is only one phase.

❌ SDLC ends after deployment.

✔ Maintenance often lasts for years.

❌ Every project uses the same SDLC model.

✔ Teams choose models based on project needs.

# Summary

The Software Development Life Cycle (SDLC) is the backbone of Software Engineering. It provides a structured approach for transforming ideas into reliable software through planning, design, development, testing, deployment, and maintenance. Every modern development methodology builds upon these core lifecycle concepts, making SDLC one of the most important topics in Software Engineering.
