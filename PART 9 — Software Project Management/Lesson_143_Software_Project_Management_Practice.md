# Introduction

Project management skills improve through practice.

A project manager must be able to:

-   Understand requirements
-   Create plans
-   Estimate resources
-   Manage risks
-   Track progress
-   Make decisions

# Practice 7 --- CPM Calculation

Activities:

  Activity   Duration
  ---------- ----------
  A          5 days
  B          10 days
  C          20 days
  D          5 days

Dependency:

    A

    ↓

    B

    ↓

    C

    ↓

    D

------------------------------------------------------------------------

# Critical Path

Since all activities are connected:

    A → B → C → D

Duration:

    5 + 10 + 20 + 5

    = 40 days

Critical Path:

    A → B → C → D

# Practice 12 --- Scenario Questions

## Question 1

Project deadline is reduced by one month.

What will you do?

Answer:

-   Analyze impact
-   Prioritize features
-   Add resources if possible
-   Use fast tracking
-   Communicate changes

------------------------------------------------------------------------

## Question 2

Budget exceeds planned cost.

Solution:

-   Identify cause
-   Review expenses
-   Control scope
-   Update stakeholders

------------------------------------------------------------------------

## Question 3

Team productivity decreases.

Solution:

-   Identify blockers
-   Improve communication
-   Provide support
-   Review workload

# Summary

Software Project Management requires practical decision-making skills.
Through planning exercises, risk analysis, scheduling problems, resource
allocation, and real-world scenarios, project managers learn how to
control uncertainty and successfully deliver software projects.
