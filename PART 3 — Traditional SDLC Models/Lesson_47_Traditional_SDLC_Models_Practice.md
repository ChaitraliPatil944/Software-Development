# Section 7 — Diagram Practice

Draw these from memory.

## Waterfall

```text
Requirements
     ↓
Design
     ↓
Development
     ↓
Testing
     ↓
Deployment
     ↓
Maintenance
```

## V-Model

```text
Requirements ↔ Acceptance Testing
System Design ↔ System Testing
HLD ↔ Integration Testing
LLD ↔ Unit Testing
        Coding
```

## Spiral

```text
Planning
   ↓
Risk Analysis
   ↓
Engineering
   ↓
Customer Evaluation
   ↺
```

## Prototype

```text
Requirements
     ↓
Prototype
     ↓

     ↓
Refinement
     ↓
Final Product
```
