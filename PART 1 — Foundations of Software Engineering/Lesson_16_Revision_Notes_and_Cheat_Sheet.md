# 1. Software Engineering

**Definition**

Software Engineering is the systematic application of engineering principles to the development, testing, deployment, and maintenance of software.

**Goal**

- Build reliable software
- Reduce cost
- Improve quality
- Support teamwork
- Simplify maintenance

---

# 2. Why Software Engineering Was Invented

Problems before Software Engineering:

- No planning
- Poor documentation
- Frequent bugs
- Missed deadlines
- Cost overruns
- Difficult maintenance

Result:

```text
Software Crisis
        ↓
Need for Engineering Practices
        ↓
Software Engineering
```
